# Русский
## Что это?
**Ресурспак**, добавляющий перевод аддонов **[Create Aeronautics](https://modrinth.com/mod/create-aeronautics)** на русский язык.

<details>
<summary>Полный список переведеных модов</summary>

[Sable](https://modrinth.com/mod/sable)

[Create Aeronautics: Dyeable Components](https://modrinth.com/mod/aeronautics-dyeable-components)

_Будет пополняться..._
</details>

## Установка
1. Скачайте любой из переведенных аддонов (необязательно, но иначе ничего не произойдет)
2. Скачайте сам ресурспак
3. Поместите архив в папку <code>resoursepacks</code>
4. Включите в самом Майнкрафте

# English
## What is this?
**Resource** pack that adds a **[Create Aeronautics](https://modrinth.com/mod/create-aeronautics)** addons translation into Russian.
<details>
<summary>Full list of translated mods</summary>

[Sable](https://modrinth.com/mod/sable)

[Create Aeronautics: Dyeable Components](https://modrinth.com/mod/aeronautics-dyeable-components)

_More to come..._
</details>

## Installation
1. Download any of the translated addons (optional, but otherwise nothing will happen)
2. Download the resource pack
3. Place the archive into the <code>resourcepacks</code> folder
4. Enable it in Minecraft itself
